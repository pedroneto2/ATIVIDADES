# BACK-END_OPA-01
back-end test for Super Opa

## How to run?

Install the dependencies and then run the app.

Commands to run in root directory:

  <strong>npm i</strong>: Install dependencies

  <strong>npm run dev</strong>: Run the app

  <strong>npm test</strong>: Run the tests with jest

  <strong>npm start</strong>: Used for deploy on Heroku

### Notes

.env MUST be added to .gitignore, but it was not added for education purposes
